# Workflow Rules

- Before implementing any Livewire or Flux pattern, use `search-docs` to verify the API
- After modifying any user-facing code, visually verify it in the browser
- Prefer concise responses — no verbose explanations unless asked
- Never run git commands without explicit user permission
- Always check CLAUDE.md for project-specific context (domain, users, locale, currency) before making product decisions
- When integrating external APIs, read official documentation (`search-docs` for Laravel ecosystem, Context7 MCP for others) — never rely on web searches or assumptions
